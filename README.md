# SQL Learning Notes

A structured collection of SQL learning notes, examples, and practice exercises completed using **MySQL**. This repository documents my progression from SQL fundamentals to more advanced database programming concepts through practical coding exercises and hands-on examples.

The repository is organised into three learning stages—**Beginner**, **Intermediate**, and **Advanced**—to reflect the progression of SQL concepts learned throughout my study.

---

# Project Overview

This repository was created to consolidate my SQL learning into a structured reference that combines concise documentation with executable SQL examples.

Rather than simply collecting code snippets, each learning stage documents:

- The SQL concepts covered
- The purpose of each topic
- Practical applications
- Key learning outcomes
- Associated SQL scripts

The exercises use the **Parks and Recreation** sample database to demonstrate SQL syntax and database operations in a consistent environment.

---

# Learning Objectives

The objectives of this repository are to:

- Build a strong foundation in SQL querying.
- Develop confidence writing increasingly complex SQL statements.
- Understand how SQL is applied in data analysis and database management.
- Practise writing clean, readable, and reusable SQL code.
- Create a structured reference for future projects and revision.
